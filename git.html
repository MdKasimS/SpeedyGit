Git Commands

Git Basics

  1. git init : will create a nw local GIT repository
  2. git clone : is used to copy a repository
  3. git add : add a file as it looks now to our next commit (stage)
  4. git commit : will create a snapshot of the changes and save it to the git directory
  5. git push : push local changes to the original

Git Branching And Merging

  1. git branch : will list your branches
  2. git checkout : switch to another branch and check it out into your working directory
  3. git merge : will merge the specified branch's history into the current branch
  4. git log : show all the commits in the current branch's history

Git Make A Change

  1. git status : list new or modified files not yet commited
  2. git diff : lists down changes and conflicts
  3. git add [file ] : stages the file, ready for the commit
  4. git reset [file ] : unstages file, keeping the file changes
  5. git commit -m "descriptive message" : commit all stages files to versioned history

Git Synchronize

  1. git remote <add> <url> : Create a new connection to a remote repo
  2. git fetch : get all changes from the origin (no merge)
  3. git pull : get all the latest changes from the origin and merge
  4. git push : is used to upload your local repository changes to the origin remote repo
